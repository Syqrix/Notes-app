Notes app
About the project

I developed this project to practice my skills. It is a simple console-based notes application, similar to what you might find on your phone. Features:

    Create, read, update, delete contacts (CRUD operations)
    Saving data to a JSON file
    Data validation

Tools

    Python: 3.12.3
    Packer manager: Uv
    Utilites: MyPy, Ruff, Pytest, Black

Instalation

How to install:

    git clone https://github.com/Syqrix/Notes-app.git
    cd notes-app
    In your IDE in console: "uv/pip install -r requirements.txt"
    Write down in your console: python/python3 main.py
